## Installation

Step 1:
Install module
```sh
npm install
```

Step 2:
Edit .env file
```sh
RMQ_USER=
RMQ_PASS=
RMQ_HOST=
RMQ_PORT=
RMQ_VHOST=
```
Step 3:
Run Project

Node:
```sh
node app
```
Or

Nodemon:
```sh
nodemon app
```
## Development Module

Amqplib (v0.6.0):
```sh
npm install amqplib --save
```
DontEnv (8.2.0):
```sh
npm install dotenv --save
```
